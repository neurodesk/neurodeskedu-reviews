# Post-Review Checklist

## Author Tasks After Review Completion

- [ ] Double check authors and affiliations (including ORCIDs)
- [ ] Make a release of the software with the latest changes from the review and post the version number here. This is the version that will be used in the JOSE paper.
- [ ] Archive the release on Zenodo/figshare and post the DOI here.
- [ ] Ensure the title and author list (including ORCIDs) in the archive match the paper.
- [ ] Ensure the license listed for the archive matches the software license.

## Editorial Tasks Prior to Merge

- [ ] Read the text of the paper and offer comments/corrections (as either a list or a PR)
- [ ] Check the references in the paper for correctness (e.g. capitalization)
- [ ] Verify the archive title, author list, version tag, and license are correct
- [ ] Verify DOI resolves correctly
- [ ] Check that all references are valid
- [ ] Approve and merge into NeurodeskEDU
